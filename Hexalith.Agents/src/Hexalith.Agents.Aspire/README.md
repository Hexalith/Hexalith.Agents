# Hexalith.Agents.Aspire

.NET Aspire hosting extensions for the Hexalith Agents module — the named extension point (AC4) for the
reusable `AddHexalithAgentsServer(...)` hosting recipe (server resource + DAPR sidecar + state-store /
pub-sub wiring).

This project is an intentionally empty shell in Story 1.1. A *functioning* AppHost/topology is explicitly
out of scope for the foundation story (AD-16; readiness-report Concern #5); the hosting extensions land
with the operational-topology stories.
