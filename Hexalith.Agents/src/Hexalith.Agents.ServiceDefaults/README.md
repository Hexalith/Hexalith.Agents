# Hexalith.Agents.ServiceDefaults

Aspire shared project (`IsAspireSharedProject=true`) — the named extension point (AC4) for the module's
`AddServiceDefaults()` host conventions: OpenTelemetry, health endpoints, service discovery, and HTTP
resilience.

Intentionally empty in Story 1.1. The conventions and their package dependencies are added by the
operational-topology story (AD-16); the foundation story only establishes the named project.
